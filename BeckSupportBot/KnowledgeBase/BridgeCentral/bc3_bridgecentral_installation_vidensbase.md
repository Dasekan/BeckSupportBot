# BridgeCentral / BC3 – Installation

## Formål

Denne vejledning beskriver de første trin i installation af BridgeCentral / BC3 på en Windows-PC samt oprettelse af klubforbindelse til bridgeklubbens database.

Vejledningen kan bruges som vidensgrundlag til supportchatbotten, når brugere spørger om installation, opstart eller klubforbindelse i BridgeCentral.

---

# Forudsætninger

Før installationen skal følgende være klar:

- en Windows-PC
- internetadgang
- en browser, fx Microsoft Edge, Google Chrome eller Firefox
- adgang til `bridge.dk`
- klubbens kode til oprettelse af klubforbindelse

---

# Installation af BridgeCentral

1. Åbn en browser på Windows-PC’en.
2. Gå til `bridge.dk`.
3. Find området, hvor BridgeCentral / BC3 kan hentes.
4. Hent installationsfilen.
5. Udpak den hentede fil, hvis den er pakket.
6. Kør installationsprogrammet.
7. Følg installationsguiden på skærmen.
8. Start BridgeCentral efter installationen.

---

# Oprettelse af klubforbindelse

Efter installationen skal der oprettes en klubforbindelse.

En klubforbindelse kobler BridgeCentral sammen med bridgeklubbens database.

## Fremgangsmåde

1. Åbn BridgeCentral.
2. Gå til forsiden af BridgeCentral.
3. Vælg **Klubforbindelser**.
4. Opret en ny forbindelse til bridgeklubbens database.
5. Angiv klubbens kode.
6. Gem forbindelsen.
7. Test at BridgeCentral kan åbne klubdata.

---

# Vigtige begreber

## BridgeCentral / BC3

BridgeCentral / BC3 er programmet, som klubben bruger til blandt andet medlemmer, klubturneringer, sektioner, Bridgemate, resultater og hjemmeside/publicering.

## Klubforbindelse

En klubforbindelse er forbindelsen mellem BridgeCentral og klubbens database.

Klubforbindelsen oprettes efter installationen og kræver normalt klubbens kode.

## Klubkode

Klubkoden bruges ved oprettelse af klubforbindelsen.  
Hvis brugeren ikke kender klubkoden, skal den findes via klubben eller erfaren support.

---

# Typiske problemer

## Brugeren kan ikke finde installationsprogrammet

Hvis brugeren ikke kan finde BridgeCentral / BC3 på `bridge.dk`, skal supporten hjælpe med at finde det korrekte downloadområde.

## Programmet kan ikke starte efter installation

Mulige årsager:
- installationen er ikke fuldført
- programmet er ikke udpakket korrekt
- Windows blokerer programmet
- der mangler nødvendige komponenter
- brugeren åbner forkert fil

## Klubforbindelsen kan ikke oprettes

Mulige årsager:
- forkert klubkode
- manglende internetforbindelse
- forkert databaseforbindelse
- manglende rettigheder
- database eller server kan ikke tilgås

## Databaseadgang fejler

Hvis BridgeCentral ikke kan få adgang til databasen, kan problemet ligge i:
- klubforbindelsen
- databaseplacering
- netværk
- adgangsrettigheder
- forkert kode eller opsætning

# Hvornår sagen bør sendes videre

Sagen bør sendes videre til erfaren support, hvis:

- brugeren ikke kan finde eller hente installationsprogrammet
- BridgeCentral ikke kan starte efter installation
- klubforbindelsen ikke kan oprettes
- der opstår fejl ved databaseadgang
- brugeren ikke ved, hvilken klubkode der skal bruges
- brugeren ikke ved, hvilken database der skal forbindes til
- fejlen handler om server, databaseplacering eller adgangsrettigheder
- der opstår en ukendt fejlmeddelelse, som ikke er dækket af vejledningen

---

# Supportflow ved installationsspørgsmål

Når en bruger spørger om installation af BridgeCentral / BC3, kan supportchatbotten følge dette flow:

1. Afklar om brugeren sidder ved en Windows-PC med internetadgang.
2. Bed brugeren åbne en browser.
3. Guide brugeren til `bridge.dk`.
4. Forklar at BridgeCentral / BC3 skal hentes fra webstedet.
5. Forklar at installationsfilen eventuelt skal udpakkes.
6. Forklar at programmet skal installeres på PC’en.
7. Forklar at BridgeCentral skal startes efter installationen.
8. Forklar at der derefter skal oprettes en klubforbindelse.
9. Forklar at klubforbindelser findes på forsiden under **Klubforbindelser**.
10. Forklar at brugeren skal angive klubbens kode.
11. Send sagen videre til erfaren support, hvis der opstår databasefejl, ukendt fejl eller usikkerhed om klubkode/database.
